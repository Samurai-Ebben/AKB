# Service images

Add one image per service to this folder. The website finds it automatically; no React changes are required.

Use one of these exact filenames with `.avif`, `.jpeg`, `.jpg`, `.png`, or `.webp`:

- `maintenance`
- `stairwell`
- `garden`
- `clearance`
- `assembly`
- `flooring`
- `drywall`
- `demolition`
- `transport`
- `technical`

Example: `garden.webp`

For consistent cards, use landscape images with a 16:9 aspect ratio. Aim for approximately 1600 x 900 pixels and keep each optimized image below 400 KB.
